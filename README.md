# Super Mini Mario

A tiny Super Mario Bros.–inspired platformer built with HTML5 canvas. Run, jump, collect coins, and stomp Goombas while working toward the flag pole.

## Running the game

Open `index.html` in any modern desktop browser. No build step or server is required.

## Controls

- Left / Right arrows – move
- Up arrow or Space – jump
- Space – start/restart when the overlay is visible

## Features

- Responsive side-scrolling camera with tile-based collision detection
- Jump buffering and coyote time for responsive controls
- Question blocks that dispense coins
- Collectible coins and score tracking HUD
- Patroling Goombas that can be stomped or cause a loss on contact
- Flag pole finish line with win screen and stats
